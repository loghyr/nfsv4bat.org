Overview

  • Where: The fall Bakeathon will be held at the Red Hat Westford Office and virtually using the NFS-VPN network.

  • When: Mon Oct 10 - Fri Oct 14

  • Event Registration: Send email to BAT Contact with "Fall Bakeathon 2022" in the subject line
    with how many engineers will be attending and the t-shirt size for each engineer.

  • Virtual Registration: following the instructions under Request To Join.
    Once a Public Key and IPv6 address are allocated, your host will be able to create an WireGuard vpn tunnel to join the network.

Testing Focus

We are going to try test as much of the protocol as possible. Here are some of the suggested technologies

  • NFSv4.2 features
  • pNFS with Flex File layouts
  • Kerberized NFS
  • NFS/RDMA on iWarp

Communications

A Discord server, called "Bake a Thon", have been created to enable participants to verbally communicate, with separate channels for different topics.

If you haven't used discord before then use the following invite link to join this server Discord BAT Server
