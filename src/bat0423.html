Testing Focus

We are going to try test as much of the protocol as possible. Here are some of the suggested technologies

  • RPC over TLS
  • NFSv4.2 features
  • pNFS with Flex File layouts
  • Kerberized NFS
  • NFS/RDMA on iWarp

Registration

We will be using a tailscale VPN to connect peer-to-peer connections on the BAT network.
To gain access to the network please register at Spring 2023 Registration.

Here are the steps to access the network:

  • Download the tailscale bits
    • Tailscale Download
  • Register with the BAT server
    • tailscale up --login-server https://headscale.nfsv4.dev
  • The registration will give an authentication link
  • Use that link to
    • Create an account on the BAT server
    • Authenticate your machine[s]
Once the tailscale interface is up, DNS will be able to find other machines
  • The command 'tailscale status' will show those machines
