Testing Focus

We are going to try test as much of the protocol as possible. Here are some of the suggested technologies

  • pNFS and NFSv4.1
  • NFSv4.2 features
  • Flex File layouts
  • Kerberized NFS
  • Nconnect
  • iWarp

Registration

To registration simply following the instructions under Request To Join. Once a Public Key and ipv6 address are allocated Your host will be able to create an OpenVPN to join the network.

The network server information can be found at Network Server Info.
