Testing Focus

We are going to try test as much of the protocol as possible. Here are some of the suggested technologies

  • NFSv4.2 features
  • pNFS with Flex File layouts
  • Kerberized NFS
  • NFS/RDMA on iWarp

Registration

To register, simply following the instructions under Request To Join. Once a Public Key and IPv6 address are allocated, your host will be able to create an OpenVPN tunnel to join the network.

A Wireguard tunnel is also available upon request.

The network server information can be found at Network Server Info.

Communications

A Discord server, called "Bake a Thon", have been created to enable attendees to interact via chat, audio, and video while testing. Informal demonstrations and presentations are organized during the afternoons.
